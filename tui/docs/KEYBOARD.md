# BC TUI Keyboard Shortcuts

## Global Navigation

| Key | Action |
|-----|--------|
| `1` | Go to Dashboard |
| `2` | Go to Agents view |
| `3` | Go to Channels view |
| `4` | Go to Costs view |
| `5` | Go to Demons view |
| `6` | Go to Processes view |
| `7` | Go to Teams view |
| `Tab` | Next tab |
| `Shift+Tab` | Previous tab |
| `?` | Show help |
| `q` | Quit / Back |

## List Navigation (Vim-style)

| Key | Action |
|-----|--------|
| `j` / `↓` | Move down |
| `k` / `↑` | Move up |
| `g` | Go to first item |
| `G` | Go to last item |
| `Enter` | Select / Open detail |

## Dashboard View

| Key | Action |
|-----|--------|
| `a` | Go to Agents |
| `c` | Go to Channels |
| `$` | Go to Costs |
| `r` | Refresh data |

## Agents View

| Key | Action |
|-----|--------|
| `j/k` | Navigate agents |
| `Enter` | View agent detail |
| `r` | Refresh |

## Agent Detail View

| Key | Action |
|-----|--------|
| `Escape` / `q` | Back to agents list |
| `r` | Refresh |

## Channels View

| Key | Action |
|-----|--------|
| `j/k` | Navigate channels |
| `Enter` | Open channel |
| `r` | Refresh |

## Channel Chat

| Key | Action |
|-----|--------|
| `i` | Enter input mode |
| `Escape` | Exit input mode |
| `j/k` | Scroll messages (navigation mode) |
| `Enter` | Send message (input mode) |
| `@` | Show mention autocomplete |
| `Tab` | Accept autocomplete suggestion |

## Costs View

| Key | Action |
|-----|--------|
| `r` | Refresh |
| `q` | Back |

## Demons View

| Key | Action |
|-----|--------|
| `j/k` | Navigate demons |
| `e` | Enable selected demon |
| `d` | Disable selected demon |
| `x` | Run selected demon |
| `r` | Refresh |
| `q` | Back |

## Processes View

| Key | Action |
|-----|--------|
| `j/k` | Navigate processes |
| `Enter` | View process logs |
| `r` | Refresh |
| `q` | Back |

## Teams View

| Key | Action |
|-----|--------|
| `j/k` | Navigate teams |
| `Enter` | Expand/collapse team |
| `r` | Refresh |
| `q` | Back |

## Input Mode

When in input mode (typing a message):

| Key | Action |
|-----|--------|
| `Enter` | Send message |
| `Escape` | Exit input mode |
| `@` + characters | Trigger mention autocomplete |
| `Tab` | Accept autocomplete |
| `↑/↓` | Navigate autocomplete suggestions |

## Tips

- Use number keys (1-7) for quick tab switching
- Press `?` anywhere for context-sensitive help
- Most views support `r` to refresh data
- Press `q` to go back or quit
