## bc cost agent

Show per-agent cost breakdown

### Synopsis

Show cost breakdown by agent. If a name is given, shows detail for that agent.

Examples:
  bc cost agent                    # All agents
  bc cost agent swift-falcon       # Specific agent

```
bc cost agent [name] [flags]
```
