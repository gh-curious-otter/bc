## bc cost daily

Show daily cost totals

### Synopsis

Show daily cost totals for the last N days.

Examples:
  bc cost daily              # Last 30 days (default)
  bc cost daily --days 7     # Last 7 days
  bc cost daily --json

```
bc cost daily [flags]
```

### Options

```
      --days int   Number of days to show (default 30)
  -h, --help       help for daily
```
