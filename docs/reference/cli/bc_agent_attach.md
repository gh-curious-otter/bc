## bc agent attach

Attach to an agent's session

### Synopsis

Attach to an agent's tmux session for direct interaction.

Use Ctrl+b d to detach and return to your shell.

Examples:
  bc agent attach eng-01   # Attach to eng-01

```
bc agent attach <agent> [flags]
```
