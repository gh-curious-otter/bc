## bc cost budget show

Show budget status

### Synopsis

Show current budget configuration and status.

Examples:
  bc cost budget show                   # Show all budgets
  bc cost budget show --agent eng-01    # Show agent budget

```
bc cost budget show [flags]
```

  -h, --help           help for show
