## bc cron show

Show cron job details

### Synopsis

Display full details of a cron job including schedule and run history stats.

Examples:
  bc cron show daily-lint
  bc cron show daily-lint --json

```
bc cron show <name> [flags]
```

      --json   Output as JSON
