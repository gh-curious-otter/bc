## bc cost dashboard

Show rich cost dashboard

### Synopsis

Show a rich formatted cost dashboard with summary, per-agent breakdown,
per-model breakdown, and budget status.

Examples:
  bc cost dashboard
  bc cost dashboard --json

```
bc cost dashboard [flags]
```
