## bc agent broadcast

Send a message to all running agents

### Synopsis

Broadcast a message to all running agents in the workspace.

Examples:
  bc agent broadcast "run tests"
  bc agent broadcast "check status"

```
bc agent broadcast <message> [flags]
```
